What is Bootstrap!

Bootstrap is a powerful free framework that makes the web development easier and faster. It is a collection of HTML, CSS and Javascript.

The Best Two Advatages of Bootstrap.
  • Responsive Mobile First Sites
  • Best Grid System which is based on 12 columns

How Can We Use Bootstrap?

Bootstrap is a free open source. Basically it is based on classes, and that would make us use less CSS. If we want to use bootsrtap, we have to first go to bootsrtap website, and then go to "getting started". We can search for anything like components, forms, layouts, etc. The code is already written there, we are allowed to copy and paste it.

Components

Bootstrap comes with an extensive components library that includes buttons and lists to carousels and modals. Components are easy to integrate into any site and require no knowledge of JavaScript to use.

Utilities

Bootstrap also offers a lot of utilities, we can control floating and clearfix, overflow, position, borders and borders colors, colors, shadows, spacing and sizing, etc

Components: Carousel.

Carousel is a slideshow that allowes us to go through content. Bootstrap provides different styles of carousel , so some would be with images only and some would be with images and text. Carousel is built with CSS and Javascipt.

Components: Buttons.

Bootstrap provides several buttons styles and sizes and each color and style conveys a specific message to the user

Be Aware of those things when using Bootstrap:

  • When working with bootstrap there will be lots of style overrides, and that will need time.
  • We won't be actually learning how to code when using bootstrap. It is just a toolkit to make our job easier and faster.
